節流 — 消費習慣
先付錢給自己
發薪日當天先自動轉帳到投資帳戶，剩下的才是生活費。不依賴月底有沒有剩。
購物車冷靜法
不要當下買，加入購物車後等 48 小時。大多數衝動消費在冷靜後會消失。
分辨好債 vs 壞債
好債：帶來資產增值。壞債：信用卡循環、BNPL。信用卡全額繳清，不跑循環利息。
警惕生活膨脹
轉職後薪資提升，消費不同步升級。增加的收入優先進投資帳戶。

財務自由第一性原理
核心公式
被動收入 ≥ 月支出 = 財務自由
邏輯鏈（閉合的）
認知提升 → 本業收入↑ → 儲蓄轉化資產 → 被動收入↑ → 覆蓋支出 → 自由
財務自由 ≠ 退休
自由的本質是選擇權，不是數字。當資產夠大，工作變成你選擇做的事，而不是你必須做的事。
郝哥四邏輯 × 個人校準
邏輯一：自由 = 選擇權 ✓
「當你不想做某件事時，你有拒絕的權利。」這是財務自由的真正定義，不是大數字。
邏輯二：資產 vs 負債 ✓
資產 = 把錢放進口袋的東西。負債 = 把錢拿走的東西。主動收入要像漏斗一樣流向資產。
邏輯三：機械式定額 ⚠ 部分修正
對一般人 100% 正確。但你對 CoWoS / HBM 供應鏈有真實認知優勢。正確框架：核心倉 70% 機械式 0050 + 衛星倉 30% 認知優勢選股，不是 100% 放棄選股。
邏輯四：投資自己是最高報酬率 ✓
早期本金小，自我投資 ROI 遠超金融投資。轉職加薪的影響遠比任何選股策略大。
三種槓桿（重要性排序）
① 財務槓桿（讓錢生錢）
股票、ETF、投資。資產規模越大複利效果越顯著。早期本金小，這個槓桿效果有限。
② 數位槓桿（代碼與媒體）
產品、內容、個人品牌。一次建立，持續產生效果。這是你中長期最重要的開源槓桿。
③ 勞力槓桿（拼命升職加薪）
天花板最低，但早期是最快提升收入的方法。轉職 = 目前最高 ROI 的財務決策。
常見執行陷阱
觀念和執行之間的鴻溝
你已經「知道」要做的事很多。財務自由是被執行出來的，不是被想出來的。
記帳 ≠ 財務管理
盲目記帳只是「事後後悔」。每月記錄四個數字就夠：收入、支出、資產淨值、儲蓄率。
副業 vs 正職的優先順序
轉職前不要分心太多副業。轉職是現在最大的槓桿，完成後再認真啟動副業試水。
